# 2026-06-11 Server wouldn't connect to iPhone hotspot (2.4 vs 5 GHz band)

**Symptom.** WiFi netplan config was correct (SSID + password right),
`wpa_supplicant` installed, yet `networkctl status wlp3s0` showed
`no-carrier (configuring)` and `Wi-Fi access point: (null)` - the card wasn't
associating with any network and got no IP.

**Cause.** The server's (my old ASUS N55S laptop) WiFi card is an **Atheros AR9285, 802.11b/g/n 2.4 GHz only**.
**iPhone Personal Hotspot broadcasts on 5 GHz by default**, so the card physically couldn't see the network.

**Fix.** On the iPhone: Settings > Personal Hotspot > enable
**Maximize Compatibility** the hotspot switches to 2.4 GHz.
